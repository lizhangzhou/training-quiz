# 新项目说明

## 项目定位

本项目以原刷题网站的稳定版本为基础，重新建立为独立的“培训刷题助手”。界面、登录、访客、云同步、错题本、收藏夹、模拟考试和手机端适配保持一致，题库已全部替换。

## 新题库统计

- 总题量：491 道；
- 单选题：303 道；
- 多选题：97 道；
- 判断题：91 道；
- 分类数量：31 个。

题库来源：

1. `老师题目考试题库.docx`；
2. `负责人题目.docx`；
3. `国网湖南电力法治企业建设与合规管理测试题.doc`。

## 题库整理规则

- 保留原题干、选项、标准答案、答案解析和依据出处；
- 判断题统一转换为“正确 / 错误”两个选项；
- 教师题库、负责人题库和法治专题分别增加来源前缀；
- 原文中没有提供解析的题目不自行补写解析；
- 法治专题题库按照原文件末尾参考答案录入。

## 独立部署

Worker 名称：

```text
training-quiz
```

D1 数据库名称：

```text
training-quiz-db
```

首次部署前必须创建新的 D1 数据库，并把真实 `database_id` 写入 `wrangler.jsonc`。

## 原文格式说明

- `国网湖南电力法治企业建设与合规管理测试题.doc` 的第 4 道多选题中，首个选项正文前缺少 `A.` 标记；项目录入时仅补齐该选项编号，选项文字和标准答案未作改动。
- 除上述选项编号格式整理外，项目没有使用外部资料补写、核对或更改标准答案。
